License & Trial System

Digital Damage tools implement a free license and trial system using $0 infrastructure. Your applications check a LICENSE.json file on GitHub, submit their license key and date, validate the trial window, and automatically lock or unlock features.

Trial System Workflow

  • Checks LICENSE.json hosted on GitHub
  • Sends license key and current date
  • Validates trial period and restrictions
  • Locks/unlocks features as required

Trial JSON Format Example

{
  "LICENSES": {
    "email@example.com": {
      "key": "DD-TRIAL-AX39F",
      "product": "AutoEX",
      "created": "2025-02-07",
      "expires": "2025-02-14",
      "status": "active"
    }
  }
}
